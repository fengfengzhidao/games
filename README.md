# 小游戏合集

一个纯前端小游戏页面，目前包含「贪吃蛇」和「宝石连连看」。项目只有一个 `index.html`，无需安装依赖，直接用浏览器打开即可游玩。

## 功能

- 贪吃蛇：方向键或 WASD 控制移动，空格暂停，支持最高分本地保存。
- 宝石连连看：连接相同宝石，连线路径最多允许两次转弯。
- 移动端适配：贪吃蛇提供屏幕方向按钮。
- 游戏切换：顶部标签可在不同游戏间切换。
- 辅助操作：连连看内置提示和洗牌次数。

## 运行

直接打开 `index.html`：

```text
index.html
```

也可以在项目目录启动任意静态服务器，例如：

```bash
npx serve .
```

然后在浏览器访问终端提示的本地地址。

## 操作说明

### 贪吃蛇

- 开始/重新开始：点击「开始」或覆盖层按钮。
- 移动：使用方向键或 WASD。
- 暂停/继续：点击「暂停」按钮，或按空格键。
- 手机操作：使用游戏下方的方向按钮。

### 宝石连连看

- 点击两颗相同宝石进行连接。
- 两颗宝石之间的路径最多只能转两次弯。
- 棋盘外圈也可以作为连线路径。
- 无可用连接时，可使用「洗牌」。
- 不确定下一步时，可使用「提示」。
